# Documentation

Design documentation and specifications for the Parallel Earth Simulator.

## Structure

- **`architecture/`** — System architecture and design
  - Core architecture overview
  - System flows and interactions
  - Design decisions

- **`data-specs/`** — Data format specifications
  - Schema definitions for seeds, reference data
  - Data validation rules
  - Import/export formats

- **`api/`** — API documentation
  - REST endpoint definitions
  - WebSocket protocols
  - Request/response schemas

- **`vision/`** — Product vision and concept
  - Core vision documents
  - Long-term roadmap

- **`era-notes/`** — Historical research notes
  - Era-specific details
  - Cultural and economic patterns
  - Historical context

## Purpose

This documentation explains:
- **How the system works** — Architecture and design
- **What data looks like** — Schemas and formats
- **How to use the API** — Endpoints and protocols
- **Why decisions were made** — Vision and rationale

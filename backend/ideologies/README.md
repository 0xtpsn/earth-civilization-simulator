# Ideologies Engine

Belief systems that drive politics, NPC behavior, and institutional actions.

## Structure

- **`political/`** — Political ideologies
  - Liberalism, conservatism
  - Nationalism, internationalism
  - Marxism, socialism
  - Fascism, authoritarianism
  - Era-bounded variants (e.g., 19th-century liberalism vs modern)

- **`economic/`** — Economic systems
  - Market capitalism
  - Feudalism
  - Socialism
  - Command economy logic
  - Mercantilism
  - Mixed economies

- **`cultural/`** — Cultural value systems
  - Collectivism vs individualism
  - Honor culture
  - Egalitarianism vs hierarchy
  - Traditionalism vs progressivism

- **`religious/`** — Religious worldviews
  - Major faith systems
  - Era-specific behavior constraints
  - Religious law and norms
  - Interfaith dynamics

- **`diffusion/`** — How ideologies spread
  - Network propagation
  - Mutation and synthesis
  - Collision and conflict
  - Faction formation
  - Institutional adoption

## Usage

NPCs and institutions reference these ideologies to:
- Decide beliefs and actions
- Form political affiliations
- Influence economic behavior
- Drive cultural change
- Create social movements

Ideologies are not static — they evolve, spread, and compete over time.
